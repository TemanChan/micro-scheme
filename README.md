# micro-scheme
Built-in keywords   
\+ \- \* /  
<   
ceiling, floor   
cons, car, cdr   
nullp, intp, doublep, symbolp, listp   
not  
print  
eval  
apply  
exit  

if: accepts 2 or 3 operands  
quote  
define  
set!  
lambda  
let  
load: e.g. (load library.scm) will load the file library.scm in the current directory.  

P.S. This interpreter uses lexical scoping.
